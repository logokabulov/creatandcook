# CREATANDCOOK

Video montaj va motion-grafika o'rganish platformasi (After Effects, Premiere Pro, DaVinci Resolve).

Til: faqat o'zbek tili.

## Tuzilma
- `index.html` — butun sayt (landing + kirish/ro'yxat + dashboard) bitta faylda.
- `vercel.json` — Vercel uchun sozlama.

## Lokal ko'rish
`index.html` faylni shunchaki brauzerda ochish kifoya.

## Deploy (Vercel)
1. Ushbu repo Vercel'ga ulanadi (Import Project).
2. GitHub'ga har push qilinganda sayt avtomatik yangilanadi.

## Keyingi bosqichlar
- Supabase: auth + ma'lumotlar bazasi + storage.
- Domen: ahost.uz orqali ulanadi.
- Dars sahifasi (video player), to'lov (Payme/Click).

## Versiya
v1 — landing, auth, dashboard (Asosiy, dasturlar, reyting, sozlamalar, obuna). Ma'lumotlar hozircha demo.
